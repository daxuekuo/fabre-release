# 更新日志

## v0.23.0 (2026-04-28)

### 新增

- 昼夜循环系统：服务器按周期自动切换昼夜
- 角色主动保存机制：在线玩家存档定期落盘，关服时全员保存
- 新增 `nightenabled` 地图标记：控制单张地图是否参与夜晚视觉同步
- 实现 `@day` 管理员指令：将服务器切换为白天
- 实现 `@night` 管理员指令：将服务器切换为夜晚

### 修复

- 修复 `@save` 设置后未立即落盘，服务器异常退出时重生点会丢失的问题

### 优化

- 重组 `conf/config.toml` 配置布局（不兼容变更）：自定义过配置的用户请参考新版重写
- 优化 `@mapflag` 管理员指令：同时给出名称和值时可直接设置该地图标记，兼容 rAthena 写法

## v0.22.0 (2026-04-27)

### 新增

- 实装 RODEX 邮件系统：游戏内邮件的收发、附件、未读通知与到期清理流程
- 实装地图标记系统：可对每张地图独立设置规则（详见 `docs/mapflags.md`）
- 新增地图标记限制类标记：`nochatroom`、`nosave`、`notrade`、`nodrop`、`nocashshop`、`norodex`
- 新增地图标记战斗与经济类标记：`pvp`、`gvg`、`bexp`
- 新增地图标记天气类标记：`snow`、`fog`、`sakura`、`leaves`、`clouds`、`clouds2`、`fireworks`
- 新增 `@setmapflag` / `@getmapflag` / `@removemapflag` 管理员指令：用于管理地图标记
- 实装快捷键栏持久化：玩家自定义的快捷键栏在登出后保留，重新登录自动恢复

- 新增 `@snow` / `@sakura` / `@clouds` 等管理员指令：切换当前地图的对应天气特效
- 新增 `@clearweather` 管理员指令：清除当前地图的全部天气特效
- 新增 `@effect <特效ID>` 管理员指令：在执行者位置播放一个客户端特效
- 新增 `@sound <文件名>` 管理员指令：在当前地图广播一个音效

### 修复

- 修复部分新版客户端 (20211117) 进入游戏后无法使用快捷键的问题
- 修复已改名宠物在进化后名字被错误重置为新形态默认名

### 优化

- 修正部分场景下宠物追击玩家时落点偏离目标的问题

### 新增

- 登录服务器：完整登录认证流程、`_M`/`_F` 后缀自动注册、客户端 EXE 哈希校验、账号状态与封禁检查、登录审计日志
- 角色服务器：角色创建与列表查看、即时删除与预约删除制、PIN 码（二次密码）系统、角色改名与槽位移动
- 地图服务器：地图加载与实体初始化、A* 寻路与基础移动系统、区域广播、Tick 同步与心跳、游戏主循环与定时器
- GM 指令系统：`@warp` 传送指令，支持自动搜索可通行位置
- 重复登录踢号：Login → Char → Map 三层踢号链路与超时保底机制
- 在线状态同步：Map → Char → Login 定期上报在线人数
- 游戏数据加载：支持 YAML 格式，递归 Import/Footer 机制，运行时热加载
- 数据库：SQLite / MySQL 双后端支持，含连接池与自动迁移
- 多语言支持：基于 Fluent 的国际化框架
- 日志系统：终端彩色输出 + 文件日志，数据加载进度条显示
- 崩溃报告：异常退出时自动生成诊断报告文件
- TOML 分层配置：支持默认配置与用户自定义配置分离加载
